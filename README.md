## Создание модели Pipeline предсказания рейтинга красного вина
##### выполнил: *Кулакова Татьяна*

### Описание проекта
Создание, обучение и сравнение моделей рейтинга красного вина; сохранение пайплайна лучшей модели:
* модель случайного леса (RandomForestRegressor)
* стекинг (StackingRegressor, базовые модели: RidgeCV, DesicionTreeRegressor, итоговая модель: RandomForestRegressor)

### Файлы проекта
* RedWine_rating_model.ipynb - *ноутбук проекта*
* pipeline_wine.pkl - *пайплайн лучшей модели*
* Red.csv - *файл данных*
* README.md - *файл описания проекта*

### Используемые инструменты
* Python 3.12.6
* pandas 2.2.3
* numpy 2.2.0
* skikit-learn 1.6.1
* matplotlib 3.10.3
* seaborn 0.13.2
* joblist 1.5.0

#### Инструкция по сборке проекта

2. Скачать файлы проекта в локальную папку
3. Запустить проект
